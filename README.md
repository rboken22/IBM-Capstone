# 🚀 SpaceX Falcon 9 Landing Prediction

## 📌 Project Overview

This project is part of the IBM Applied Data Science Capstone. The objective is to predict whether the **SpaceX Falcon 9 first stage** will land successfully using machine learning techniques.

Since SpaceX reuses Falcon 9 boosters, predicting landing success can help estimate launch costs and provide insights for competitors in the aerospace industry.

The project includes:

- Data Collection using SpaceX API & Web Scraping
- Data Wrangling & Preprocessing
- Exploratory Data Analysis (EDA)
- Interactive Visualizations
- Machine Learning Modeling
- Predictive Analysis
